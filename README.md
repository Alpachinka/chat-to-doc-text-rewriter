# ChatToDoc — LaTeX → Word Converter

> Конвертуйте текст із формулами LaTeX від чат-ботів (ChatGPT, Claude, Gemini) у формат, який ідеально вставляється в Microsoft Word як редаговані рівняння.

**🔗 Живий сайт:** https://alpachinka.github.io/chat-to-doc-text-rewriter/

---

## Як запустити на GitHub Pages (безкоштовно)

### Крок 1 — Клонуйте або завантажте файли
Репозиторій вже містить три файли:
- `index.html`
- `styles.css`
- `app.js`

### Крок 2 — Увімкніть GitHub Pages

1. Відкрийте репозиторій на GitHub: https://github.com/Alpachinka/chat-to-doc-text-rewriter
2. Перейдіть у **Settings** (вкладка вгорі праворуч)
3. У лівому меню знайдіть **Pages**
4. У розділі **Branch** оберіть `main` і папку `/ (root)`
5. Натисніть **Save**

GitHub через 1–2 хвилини опублікує сайт за адресою:
```
https://alpachinka.github.io/chat-to-doc-text-rewriter/
```

---

## Як користуватись

1. **Вставте текст** із ChatGPT, Claude або Gemini у поле ліворуч
2. **Налаштуйте шрифт** і розмір (Calibri 11pt, Arial 12pt тощо) щоб відповідало вашому документу Word
3. **Натисніть «Скопіювати для MS Word»** — текст разом із формулами у форматі MathML буде скопійовано
4. **Відкрийте Word** і натисніть `Ctrl+V` — формули перетворяться на рідні рівняння Word!

> **Підказка:** Якщо формули показуються як код, одразу після вставки натисніть `Ctrl` → оберіть "Зберегти лише текст", або вставте у поле рівняння (`Alt` + `=`).

## Підтримувані формати LaTeX

| Формат | Приклад |
|--------|---------|
| Рядкова формула | `$x^2 + y^2 = z^2$` |
| Блочна формула | `$$\sum_{i=1}^{n} x_i$$` |
| Рядкова (альт.) | `\( e^{i\pi} + 1 = 0 \)` |
| Блочна (альт.) | `\[ \frac{a}{b} \]` |

## Технічний стек

- **HTML5 / Vanilla CSS / Vanilla JS** — без фреймворків
- **[KaTeX](https://katex.org/)** — рендеринг LaTeX та генерація MathML
- **[Marked.js](https://marked.js.org/)** — парсинг Markdown
- **[Lucide Icons](https://lucide.dev/)** — іконки
- **GitHub Pages** — безкоштовний HTTPS-хостинг

---

## Чому GitHub Pages?

Сучасний браузерний API для роботи з буфером обміну (`navigator.clipboard.write`) вимагає **HTTPS**-з'єднання. GitHub Pages автоматично надає захищений протокол, що дозволяє кнопці «Скопіювати» коректно працювати з HTML-вмістом (включно з формулами MathML).
